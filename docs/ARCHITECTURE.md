## Constitutional Architecture (L0-L7)

GNCE implements an 8-layer constitutional governance system:

### Phase I: Constitutional Decision (L0-L4)
- **L0**: Pre-execution validation
- **L1**: Verdict & constitutional outcome  
- **L2**: Deterministic decision lineage
- **L3**: Deterministic rule engine (DRE)
- **L4**: Policy lineage & authority

### Phase II: Enforcement & Monitoring (L5-L7)
- **L5**: Cryptographic execution token (CET)
- **L6**: Behavioral drift detection
- **L7**: Veto path & execution feedback

### GPUx Hardware Readiness
The system includes a Hardware Abstraction Layer (HAL) for future GPUx hardware execution.
